# 智慧安全隐患举报系统 - 微信小程序版本

这是一个基于微信小程序的智慧安全隐患举报系统，用于工程项目中的安全隐患上报、处理和统计分析。

## 项目结构

```
safe/
├── app.js                 # 小程序入口文件
├── app.json              # 小程序配置文件
├── app.wxss              # 全局样式文件
├── project.config.json   # 项目配置文件
├── sitemap.json          # 搜索索引配置文件
├── pages/                # 页面目录
│   ├── index/            # 首页 - 标段选择
│   ├── section/          # 标段角色选择
│   ├── main/             # 主界面
│   ├── report/           # 举报页面
│   ├── detail/           # 详情页面
│   ├── stats/            # 统计页面
│   └── my-reports/       # 我的举报页面
├── components/           # 自定义组件目录
├── utils/                # 工具函数目录
└── images/               # 图片资源目录
```

## 功能特性

### 用户角色
- **员工**: 可以上报安全隐患、查看个人举报记录
- **安全环保部**: 可以处理举报、查看统计数据、管理安全知识

### 主要功能
1. **标段选择**: 支持多标段项目管理
2. **隐患举报**: 支持拍照上传、位置定位、详细描述
3. **处理流程**: 提交 → 待处理 → 处理中 → 已办结
4. **统计分析**: 隐患类型分布、解决率统计
5. **安全知识库**: 安全知识学习和文档管理
6. **通知系统**: 实时通知和消息提醒

### 隐患类型
- 消防隐患
- 用电隐患
- 设备隐患
- 高处作业
- 临边防护
- 环境保护
- 个人防护装备
- 其他隐患

## 开发说明

### 环境要求
- 微信开发者工具
- 微信小程序基础库 2.19.4+

### 安装和运行
1. 使用微信开发者工具打开项目
2. 配置项目信息
3. 点击编译运行

### 数据存储
当前版本使用内存存储模拟数据，生产环境需要对接后端API。

## 页面说明

### 首页 (pages/index)
- 标段选择界面
- 项目信息展示

### 标段选择 (pages/section)
- 身份选择（我要举报/个人中心/安全环保部）
- 安全知识入口

### 主界面 (pages/main)
- 员工界面：快速举报、统计卡片、最近举报
- 安全环保部界面：待处理/处理中/已办结举报列表

### 举报页面 (pages/report)
- 隐患类型选择
- 严重程度设置
- 照片上传
- 位置信息
- 详细描述

### 详情页面 (pages/detail)
- 举报详情展示
- 处理流程时间线
- 照片查看
- 处理操作（安全环保部）

### 统计页面 (pages/stats)
- 数据统计卡片
- 时间筛选
- 图表展示

### 我的举报 (pages/my-reports)
- 个人举报记录
- 状态筛选
- 安全知识管理（安全环保部）

## 技术特点

- 使用微信小程序原生框架开发
- 响应式设计，适配不同屏幕尺寸
- 模块化组件设计
- 数据驱动的界面更新
- 丰富的交互效果和动画
